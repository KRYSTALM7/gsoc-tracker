Deploy Guide
Get this app live in under 10 minutes — step by step
What you need: A GitHub account (you have KRYSTALM7) and this HTML file. That's it. No backend, no server, no cost.
Option A — GitHub Pages (Free)
1
Create a new repository
Go to github.com/new  ·  Name it: gsoc-tracker  ·  Set to Public  ·  Click "Create repository"
2
Upload this file
In the repo, click "Add file" → "Upload files" → drag and drop index.html → Commit changes
3
Enable GitHub Pages
Go to repo Settings → Pages → Source: "Deploy from branch" → Branch: main → folder: / (root) → Save
4
Your live URL
After 1-2 minutes your site will be live at:
https://KRYSTALM7.github.io/gsoc-tracker
Option B — Netlify (Faster, Drag & Drop)
1
Sign up / Log in
Go to netlify.com and sign up with your GitHub account KRYSTALM7
2
Deploy by drag & drop
On your Netlify dashboard, drag the index.html file directly into the deploy zone. It goes live instantly.
3
Get your URL
Netlify gives you a random URL like random-name.netlify.app — you can rename it in Site Settings → Domain Management
4
Update anytime
Just drag and drop the updated file again. Or connect to your GitHub repo for automatic deploys on every push.
About the Login System
Your credentials and all data are stored in your browser's localStorage. This means:

• Data persists across sessions on the same browser
• No server, no database — completely private
• To sync across devices, use the Export / Import feature
• Password is hashed before storing (SHA-256)
Data Management
Export your data before switching browsers or devices. Import restores everything exactly as it was.
